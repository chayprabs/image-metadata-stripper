# Contributing

Thank you for contributing to ExifScrub.

## Development

1. Fork and clone the repository.
2. Install Node 22+ and pnpm 9+.
3. Run `pnpm install` and `pnpm dev`.
4. For worker changes, run `docker compose up` or install ExifTool locally.

## Pull requests

- Keep changes focused.
- Run `pnpm run typecheck`, `pnpm test`, and `pnpm build` before submitting.
- Follow conventional commit messages.

## Security

See [SECURITY.md](SECURITY.md) for reporting vulnerabilities.
